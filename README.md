# SwiftSync Specification

_SwiftSync_ is a protocol [originally proposed by Ruben Somsen](https://gist.github.com/RubenSomsen/a61a37d14182ccd78760e477c78133cd) to accelerate initial block download. These documents define the data structures and algorithms defined in the proposal.
- `swiftsync`: The protocol for IBD
- `hintsfile`: A concise representation of the UTXO set
- `block_undo`: Input data for Bitcoin blocks
